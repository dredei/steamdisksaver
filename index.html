Steam Disk Saver

Recover disk space and improve game startup time by deleting unnecessary files.

Download preview

Recover disk space

Most games ship with files that aren't accessed by the game at all. These remain on your disk, wasting space, never to be used. Steam Disk Space lets you delete these files, recovering disk space.

Improve game startup time

Some games show videos on startup, like intro movies or developer, publisher and hardware logos. Sometimes, these videos aren't even skippable. Steam Disk Saver allows you to delete these videos, making the game start up faster.

Safe to use

By design, Steam Disk Saver can only delete files within a game's directory. It will never delete anything else. All supported games have been tested to work properly after files have been deleted.

If needed, Steam Disk Saver allows you to tell Steam to restore deleted files.

Skip DirectX installs

Never wait for Steam to install DirectX yet again. Steam Disk Saver allows you to delete redistributables before running a game for the first time.

If it turns out that the game doesn't run because of a missing installation, you can restore deleted files. Steam will install the redistributables next time you start the game.

Automatic definition update

Steam Disk Saver always obtains the latest definition when started. This way, you don't have to download Steam Disk Saver again when new definitions for games are released.

Unknown games or bugs

If you find any bugs, encounter a game reported as 'Unknown game' by Steam Disk Saver or find unnecessary files which aren't deleted, please post in the Steam group. If you're a programmer, feel free to fork the repository and submit pull requests!

System requirements

Steam Disk Saver only requires .NET Framework 4. This is most likely already installed on your system.

Limitations

Steam Disk Saver only supports games using the current .acf metadata format. Games using the .ncf or .gcf formats are not supported.

Game developers

If Steam Disk Saver causes any kind of problem for your game, please contact me through email (or create an issue on the issue tracker) so we can work together to resolve the problem.
